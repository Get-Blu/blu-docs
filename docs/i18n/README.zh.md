## 这是什么仓库？

**`blu-docs`** 是 **Blu 文档网站** 的所在地。

此仓库包含用于构建 **Blu VS Code 扩展** 的官方文档和网站的所有内容和代码。

我们特意将其与主扩展仓库**分开**，以保持简单、清晰和易于维护。

 在寻找扩展本身吗？  
**https://github.com/Get-Blu/blu**

---

## 为所有人而建（特别是初学者）

此文档网站仅使用：

- **HTML**
- **CSS**
- **原生 JavaScript**

没有框架。  
没有构建工具。  
没有复杂设置。

这意味着：
-  **新程序员可以轻松贡献**
-  您不需要 React、Vue 或任何 JS 框架知识
-  您可以直接打开文件、编辑并立即查看结果
-  非常适合首次开源贡献

如果您了解基本的 Web 基础知识，就可以帮助改进 Blu 的文档。

## 如何在本地运行

此文档网站使用**纯 HTML、CSS 和 JavaScript** 构建，因此本地运行非常简单。

**没有依赖项**，**没有构建步骤**，也**不需要命令行设置**。

### 步骤

1. 将此仓库克隆到本地计算机
2. 打开项目文件夹
3. 双击 `index.html`  
   或右键单击并在任何现代 Web 浏览器中打开

就这样！  
网站将立即加载。

### 可选（获得更好的编辑体验）

如果您经常编辑文档，可以：
- 在 **VS Code** 中打开项目
- 使用 **Live Server** 扩展实现自动重新加载

这完全是可选的 — 项目在没有它的情况下也能完美运行。
