## ¿Qué es este repositorio?

**`blu-docs`** es el hogar del **sitio web de documentación de Blu**.

Este repositorio contiene todo el contenido y código utilizado para construir la documentación y el sitio web oficial de la **extensión Blu para VS Code**.

Se mantiene **separado** intencionalmente del repositorio principal de la extensión para mantener las cosas simples, limpias y fáciles de mantener.

 ¿Buscas la extensión en sí?  
**https://github.com/Get-Blu/blu**

---

## Construido para todos (especialmente principiantes)

Este sitio de documentación está construido usando **solamente**:

- **HTML**
- **CSS**
- **JavaScript Vanilla**

Sin frameworks.  
Sin herramientas de compilación.  
Sin configuraciones complejas.

Esto significa:
-  **Los nuevos programadores pueden contribuir fácilmente**
-  No necesitas conocimiento de React, Vue o ningún framework de JavaScript
-  Puedes abrir archivos directamente, editarlos y ver resultados inmediatamente
-  Perfecto para primeras contribuciones al código abierto

Si conoces los fundamentos básicos de la web, puedes ayudar a mejorar la documentación de Blu.

## Cómo ejecutarlo localmente

Este sitio web de documentación está construido usando **HTML, CSS y JavaScript plano**, por lo que ejecutarlo localmente es intencionalmente simple.

**No hay dependencias**, **ni pasos de compilación**, y **no se requiere configuración por línea de comandos**.

### Pasos

1. Clona este repositorio en tu máquina local
2. Abre la carpeta del proyecto
3. Haz doble clic en `index.html`  
   o haz clic derecho y ábrelo en cualquier navegador web moderno

¡Eso es todo!  
El sitio se cargará inmediatamente.

### Opcional (para una mejor experiencia de edición)

Si editas la documentación con frecuencia, puedes:
- Abrir el proyecto en **VS Code**
- Usar la extensión **Live Server** para recargas automáticas

Esto es completamente opcional — el proyecto funciona perfectamente sin ello.
