Introducing blu v1.0 - Your AI coding assistantRELEASE

We're excited to launch blu version 1.0. Here's what it does and why we built it.

+

Hey everyone! We're really excited to share blu v1.0 with you today. After months of work, it's finally ready for you to try.

What is blu?

blu is a VS Code extension that brings AI right into your editor. Think of it as having a smart coding buddy who understands your entire project and can help you write code, fix bugs, and build features faster.

What can it do?

Here's the cool stuff blu can help you with:

Understands your whole codebase
blu reads and understands your entire project, not just the file you're working on. So when you ask it to add a feature, it knows where things are and how they connect.

Writes actual code
Just tell blu what you want in plain English. Need a login form? A database query? A React component? blu will write it for you.

Fixes bugs automatically
Copy an error message, paste it to blu, and it'll find the problem and suggest a fix. Most of the time, it gets it right on the first try.

Works with any AI model
Use Claude, GPT-4, or even run models locally on your computer. You choose what works best for you.

Why we built this

Honestly? We were frustrated with existing tools. They were either too complicated, too expensive, or didn't work well with big projects. We wanted something simple that just worked.

Plus, we care about privacy. blu doesn't send your code to our servers. Everything stays on your machine. Your code is yours.

It's free and open source

You can use blu completely free. And because it's open source, you can see exactly how it works. No hidden stuff, no sneaky data collection.

If you're a developer who wants to check the code or contribute, everything's on GitHub.

What's next?

This is just version 1.0. We've got tons of ideas for making blu even better:

• Better understanding of really large projects
• Automatic code reviews and suggestions
• Team features for sharing prompts and workflows
• Git integration for better commit messages

But we need your feedback to know what matters most. Try it out and let us know what you think!

Get started

Download blu from the VS Code marketplace, add your AI API key (we'll guide you through it), and start coding smarter.

We can't wait to see what you build with blu.

Thanks for checking it out!
- Garv Agnihotri
