# Contributing to Blu Docs

Thanks for your interest in contributing to **Blu Docs**  
All contributions are welcome — especially from beginners.

## What you can help with

- Fix typos or improve wording
- Improve existing documentation
- Add new docs or examples
- Improve HTML / CSS / JavaScript
- Add or improve translations (`docs/i18n/`)
- Add comments

Small changes matter 

## How to contribute

1. Fork this repository
2. Create a new branch
3. Make your changes
4. Open a Pull Request

That’s it.

## Local setup

This project uses **plain HTML, CSS, and JavaScript**.

- No dependencies
- No build tools
- No setup required

Just open `index.html` in your browser.

## Guidelines

- Keep things simple and readable
- Follow the existing file structure
- Be respectful and kind in discussions

## Questions or ideas?

- Open a **GitHub Discussion**
- Or open an **Issue**

Thanks for helping make Blu better
