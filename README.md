## What is this repository?

**`blu-docs`** is the home of **Blu’s documentation website**.

This repository contains all the content and code used to build the official docs and website for the **Blu VS Code extension**.

It is intentionally kept **separate** from the main extension repository to keep things simple, clean, and easy to maintain.

 Looking for the extension itself?  
**https://github.com/Get-Blu/blu**

---

## Built for everyone (especially beginners)

This documentation site is built using **only**:

- **HTML**
- **CSS**
- **Vanilla JavaScript**

No frameworks.  
No build tools.  
No complex setup.

This means:
-  **New programmers can contribute easily**
-  You don’t need React, Vue, or any JS framework knowledge
-  You can open files directly, edit them, and see results immediately
-  Perfect for first open-source contributions

If you know basic web fundamentals, you can help improve Blu’s docs

## How to run locally

This documentation website is built using **plain HTML, CSS, and JavaScript**, so running it locally is intentionally simple.

There are **no dependencies**, **no build steps**, and **no command-line setup** required.

### Steps

1. Clone this repository to your local machine
2. Open the project folder
3. Double-click `index.html`  
   or right-click and open it in any modern web browser

That’s it   
The site will load immediately.

### Optional (for a better editing experience)

If you’re editing the docs frequently, you can:
- Open the project in **VS Code**
- Use the **Live Server** extension for automatic reloads

This is completely optional — the project works perfectly without it.
